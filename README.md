# ⚽ ScoreHub - Football Live Score Website

A modern football web application built with React.js and Node.js.

## Features

* Live Football Scores
* Match Statistics
* Team Details
* Player Profiles
* League Standings
* Top Scorers
* Football News
* Multi-language Support
* Dark / Light Mode
* Favorite Teams & Matches

## Tech Stack

### Frontend

* React.js
* Vite
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js
* REST APIs
